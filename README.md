## Features

1. User authentication with Firebase (Email/Password & Google login)  
2. Protected routes for logged-in users  
3. Add, update, and delete jobs/tasks  
4. Accept jobs as a freelancer  
5. View your posted and accepted jobs  
6. Display latest jobs dynamically on the homepage  
7. Filter and sort jobs by category or date  
8. Light/Dark theme toggle  
9. Fully responsive design for mobile, tablet, and desktop  
10. SweetAlert2 notifications for success/error messages  
11. Custom 404 error page for invalid routes  
12. Secure environment variable management for Firebase and MongoDB  
13. Navbar dynamically updates based on login status  
14. Image preview for job posts  
15. Smooth UI animations using Framer Motion or similar libraries
